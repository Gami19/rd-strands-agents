## 5. 質問の使い方

### ADS 前の事前配布

- Q1〜Q32 を事前アンケートとして参加者に配布
- 回答は ADS 当日までに収集
- 未回答の質問は ADS セッション中に深掘り

### ADS セッション中の進め方

1. **オープニング（30分）**: プロジェクトの目的と ADS の進め方を説明
2. **発見フェーズ（2-3時間）**: Q1〜Q32 の回答をレビューし、深掘り
3. **ホワイトボーディング（2-3時間）**: As-Is / To-Be のアーキテクチャ図を作成
4. **候補評価（1-2時間）**: アーキテクチャ候補の比較・議論
5. **クロージング（30分）**: 次のステップとフォローアップの確認
