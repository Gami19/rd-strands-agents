# 技術SEO監査チェックリスト

技術SEO監査の網羅的チェックリスト。優先度順に分類し、各項目に合格基準と確認方法を記載する。

- 2. インデックス管理（10項目）

## 1. クローラビリティ

### robots.txt

| # | 確認項目 | 合格基準 | 確認方法 |
|:---|:---|:---|:---|
| 1 | robots.txt が存在する | `/robots.txt` にアクセス可能 | URL直接アクセス |
| 2 | 重要ページがブロックされていない | `/`, `/blog/`, `/products/` 等が `Allow` | robots.txt の内容確認 |
| 3 | サイトマップ参照がある | `Sitemap:` ディレクティブが記載 | robots.txt の末尾確認 |
| 4 | AIボットが許可されている | GPTBot, PerplexityBot, ClaudeBot が `Disallow` されていない | robots.txt 確認 |

### XML サイトマップ

| # | 確認項目 | 合格基準 | 確認方法 |
|:---|:---|:---|:---|
| 5 | サイトマップが存在する | `/sitemap.xml` にアクセス可能 | URL直接アクセス |
| 6 | 形式が正しい | XML スキーマに準拠 | XML バリデーター |
| 7 | 正規URLのみを含む | noindex/リダイレクトURLを含まない | サイトマップクローラー |
| 8 | 最終更新日が正確 | `<lastmod>` が実際の更新日と一致 | サイトマップの内容確認 |
| 9 | Search Console に提出済み | 送信・インデックス状況が表示される | Search Console |

### サイト構造

| # | 確認項目 | 合格基準 | 確認方法 |
|:---|:---|:---|:---|
| 10 | クリック深度が適切 | 重要ページがホームから3クリック以内 | クローラーツール |
| 11 | 論理的な階層構造 | カテゴリ → サブカテゴリ → 詳細の構造 | サイトマップ・ナビゲーション確認 |
| 12 | 孤立ページがない | 全ページに内部リンクが存在 | クローラーツール |

### クロールバジェット（大規模サイト向け）

| # | 確認項目 | 合格基準 | 確認方法 |
|:---|:---|:---|:---|
| 13 | パラメータ URL が制御されている | 不要なパラメータ URL が canonical で正規化 | URL パラメータ分析 |
| 14 | ファセットナビゲーションの制御 | フィルタ組み合わせによる大量URL生成を抑制 | クローラーツール |

## 2. インデックス管理

| # | 確認項目 | 合格基準 | 確認方法 |
|:---|:---|:---|:---|
| 15 | インデックス数が期待値と一致 | `site:domain.com` の結果数 ≒ サイトマップURL数 | Google 検索 |
| 16 | canonical タグが全ページにある | `<link rel="canonical">` が存在 | HTML ソース確認 |
| 17 | 自己参照 canonical が正しい | ユニークページは自分自身を参照 | クローラーツール |
| 18 | 重複ページに canonical 設定 | HTTP/HTTPS, www/非www, 末尾スラッシュの統一 | クローラーツール |
| 19 | noindex の意図確認 | 重要ページに誤って noindex がない | Search Console |
| 20 | リダイレクトチェーンなし | リダイレクトが1ホップ以内 | リダイレクトチェッカー |
| 21 | リダイレクトループなし | 循環リダイレクトが存在しない | クローラーツール |
| 22 | ソフト404なし | 存在しないURLが正しく404を返す | HTTP ステータスコード確認 |
| 23 | hreflang 設定（多言語サイト） | 言語・地域ごとに正しい hreflang タグ | hreflang バリデーター |
| 24 | ページネーション処理 | `rel="next"/"prev"` または適切な代替手法 | HTML ソース確認 |

## 3. Core Web Vitals・パフォーマンス

| # | 確認項目 | 合格基準 | 確認方法 |
|:---|:---|:---|:---|
| 25 | LCP（Largest Contentful Paint） | < 2.5秒 | PageSpeed Insights |
| 26 | INP（Interaction to Next Paint） | < 200ms | PageSpeed Insights |
| 27 | CLS（Cumulative Layout Shift） | < 0.1 | PageSpeed Insights |
| 28 | TTFB（Time to First Byte） | < 800ms | WebPageTest |
| 29 | 画像の最適化 | WebP/AVIF 形式、適切なサイズ | PageSpeed Insights |
| 30 | 画像の遅延読み込み | ファーストビュー外の画像に `loading="lazy"` | HTML ソース確認 |
| 31 | レスポンシブ画像 | `srcset` と `sizes` で適切なサイズ配信 | HTML ソース確認 |
| 32 | JavaScript の最適化 | 不要なJSの遅延読み込み・分割 | DevTools Coverage |
| 33 | CSS の配信最適化 | クリティカルCSS のインライン化 | PageSpeed Insights |
| 34 | CDN の利用 | 静的アセットがCDN 経由で配信 | ネットワークタブ確認 |

## 7. オンページSEO

| 54 | ブランド名 | 末尾に「| ブランド名」形式で配置 | 目視確認 |

| 56 | 文字数 | 120-160文字 | 文字数カウント |
| 57 | 行動喚起を含む | クリックを促す表現がある | 目視確認 |

## 推奨ツール

| ツール | 種別 | 用途 |
|:---|:---|:---|
| Google Search Console | 無料 | インデックス状態・検索パフォーマンス |
| PageSpeed Insights | 無料 | Core Web Vitals・パフォーマンス |
| Google Rich Results Test | 無料 | 構造化データ検証 |
| Bing Webmaster Tools | 無料 | Bing インデックス・Copilot 対応 |
| Screaming Frog | 有料 | サイト全体クロール・技術監査 |
| Ahrefs / Semrush | 有料 | キーワード・競合・バックリンク分析 |
| Sitebulb | 有料 | 視覚的な技術SEO監査 |
