# 業種別 パフォーマンスベンチマーク

> 広告クリエイティブの効果測定に使える業種別ベンチマーク集。目標設定と改善判断の基準として活用する。

---

## 指標の定義

| 指標 | 計算式 |
|:---|:---|
| **CTR**（クリック率） | クリック数 / インプレッション数 × 100 |
| **CVR**（コンバージョン率） | コンバージョン数 / クリック数 × 100 |
| **CPA**（獲得単価） | 広告費 / コンバージョン数 |
| **CPC**（クリック単価） | 広告費 / クリック数 |

## 改善目標の設定方法

### ステップ 1: ベースラインを計測する

最低 1,000 imp（理想は 5,000+）のデータで現状値を算出する。

### ステップ 2: ギャップを判定する

| 現状値の位置 | 判定 | アクション |
|:---|:---|:---|
| 平均未満 | **要改善** | クリエイティブの抜本的見直し |
| 平均〜良好 | **標準** | ABテストで段階的改善 |
| 良好〜優秀 | **好調** | バリエーション拡大でスケール |
| 優秀超え | **トップ** | 勝ちパターンを他キャンペーンに横展開 |

### ステップ 3: 3フェーズで段階的に改善する

- **フェーズ 1**（1-2週間）: 現状値 → 業種平均。低パフォーマンス広告の停止、基本改善
- **フェーズ 2**（3-4週間）: 平均 → 良好ライン。訴求軸ABテスト、ヘッドラインの多様化
- **フェーズ 3**（5-8週間）: 良好 → 優秀ライン。勝ちパターンの拡大、微調整の反復

### ステップ 4: 指標別の改善レバー

| 指標 | 改善レバー | 期待改善率 |
|:---|:---|:---|
| **CTR** | ヘッドラインの訴求軸変更 | +20 - 50% |
| **CVR** | ディスクリプション + LP 整合性 | +10 - 25% |
| **CPA** | CTR・CVR 同時改善 + 入札最適化 | -15 - 30% |
| **CPC** | 品質スコア / 関連度スコア向上 | -10 - 20% |

改善サイクル: Google RSA 2-4週間 / Meta 1-2週間 / LinkedIn 3-4週間

## 統計的有意性の判断

| 指標 | 最低サンプル | 推奨サンプル |
|:---|:---|:---|
| CTR 比較 | 1,000 imp | 5,000 imp |
| CVR 比較 | 100 クリック | 500 クリック |
| CPA 比較 | 30 CV | 100 CV |

**判断を急いではいけないケース**: imp 1,000未満 / 配信開始72時間以内 / 曜日偏り（1週間未満） / 季節要因

## ベンチマーク活用の注意点

1. **数値は参考値**: 同じ業種でもターゲット粒度・価格帯・地域で大幅に変動する。海外データの直接適用は避ける
2. **前提を揃える**: キャンペーン目的（認知 vs リード vs 購入）、CV定義（資料DL vs 購入完了）、計測期間、デバイスで指標の意味が変わる
3. **自社データ最優先**: 業界ベンチマークより自社の過去データ（前月比・前年同月比）を信頼すべき
4. **単一指標に囚われない**: CTR高+CVR低は「クリックベイト」。最終的には ROAS/ROI で総合判断
