# プロンプトテンプレート集

> 自習用・メンター面談用のプロンプトテンプレート。新人が自らオンボーディングスキルを活用するための定型表現集。

## 自習用プロンプトテンプレート（25個）

## 状況別プロンプトの使い分けガイド

### 入社初日

```
推奨プロンプト順序:
1. N-1「オンボーディングを始めたいです」
2. N-3「学習ロードマップを見せてください」
3. N-5「Culture カテゴリの概要を教えてください」
4. N-6「企業理念の要点を教えてください」
```

### 1つのカテゴリを完了した時

```
推奨プロンプト順序:
1. Q-1「[カテゴリ] の基礎クイズを出してください」
2. N-4「今の進捗を確認したいです」
3. N-2「次に何を学べばいいですか？」
```

### 1on1 の前日

```
推奨プロンプト順序:
1. 1-1「1on1 の事前準備シートを作ってください」
2. K-4「今週の KPT をまとめてください」
3. S-1「来週の SMART 目標を設定したいです」
4. 1-2「相談トピックの優先順位をつけてください」
```

### 不安やストレスを感じた時

```
推奨プロンプト:
- M-5「[不安の内容] について相談したいです」
- M-6「キャリアについて相談したいです」
→ 必要に応じて人間のメンターへの接続を推奨
```

## プロンプトのカスタマイズガイド

| ポイント | 説明 | 例 |
|:---|:---|:---|
| **具体的に** | 抽象的な質問より具体的な質問の方が良い回答を得られる | ×「わからない」→ ○「NDA の適用範囲がわからない」 |
| **文脈を添える** | 状況を説明すると的確なアドバイスが得られる | ○「Tech の開発フローを読んだが、PR のルールが理解できない」 |
| **目的を明示** | 何のために聞くのかを伝える | ○「来週の 1on1 で話すために、今週の進捗を整理したい」 |
| **範囲を指定** | クイズの範囲や難易度を指定する | ○「Biz カテゴリの応用クイズを 5 問出してください」 |
